# 数据表操作相关

## 获取组织下所有表

```bash
GET /api/{organization}/streams
```

## 获取组织下所有表

```bash
GET /api/{organization}/streams
```

## 获取表的schema

```bash
GET /api/{organization}/{stream}/schema
```

## 获取表的属性

```bash
POST /api/{organization}/{stream}/settings
```

## 删除表

```bash
DELETE /api/{organization}/{stream}
```
